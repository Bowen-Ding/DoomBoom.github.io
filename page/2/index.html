Hexo搭建个人博客及多终端同步使用

  1. 前言

    前期准备:

    a. git账号及git for windows客户端

    b. node.js安装并添加到系统环境变量

  2. 搭建github博客

    2.1 创建仓库

    ​ 新建一个名为 git用户名.github.io 的仓库,这也是博客的访问地址

    2.2 绑定域名

    ​ a. 先到云服务商如阿里云购买一个域名,根据域名带不带www配置方式也会有所不同

    ​ b. 若不带www,需要先ping一下git用户名.github.io 的ip,然后到域名DNS设置页,将A记录指向你ping出来的IP，将CNAME指向git用户名.github.io，这样可以保证无论是否添加www都可以访问.

    ​ c. 然后到github项目根目录新建一个名为CNAME的文件，文件内容填写上域名

    ​ 注意: git用户名.github.io 这个地址仍然可以访问,将会跳转到域名

  3. 配置SSH public key

    a. 打开git客户端的git bash,执行ssh-keygen

    b. cat ~/.ssh/id_rsa.pub 命令查看public key,登录github,依次进入settings -> SSH and GPG keys -> new SSH key,然后将public key粘贴进去

    4.4 写博客

    ​ a. 执行hexo new "博客标题" 命令,可以在source/_post下面找 博客标题.md文件,在这个文件中编辑博客正文

    ​ b. 然后执行hexo g ,hexo d 完成博客发布,登录 git用户名.github.io 即可查看

    ​ c. 若出现错误可以通过 hexo clean 修复,防止意外建议先对关键文件进行备份
