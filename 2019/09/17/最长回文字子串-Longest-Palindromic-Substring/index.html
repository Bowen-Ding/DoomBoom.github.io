最长回文字子串(Longest Palindromic Substring)

algorithm
  • algorithm
  • leetcode

leetcode 第2题

  1. 题目描述

  2. 解题思路

    2.1 中心扩展法

    ​ 遍历字符串,假设当前字符是回文字符串的中心,然后向外扩散,分为2种情况:1. 中心为单字符;2. 中心为2个相等字符。

    2.2 动态规划

    ​ 假设当前字符串为回文字符串,则其必然满足左边界等于右边界,且去掉2个边界依然为回文字符串,以此建立状态转移方程
