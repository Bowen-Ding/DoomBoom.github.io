algorithm
  • algorithm
  • leetcode

leetcode 第2题

  1. 题目描述

  2. 解题思路

    a. 解法 回溯法: 运用递归,逐个匹配s的字符

    b. 解法 动态规划: 用一个二维数组保存 s[i:] 与 p[j:] 的匹配结果,返回s[0:] 与 p[0:] 的;这题动态规划有两种思路,一种从第一个字符往最后一个字符匹配,另一种从最后一个字符开始倒推,用java和python各实现一种

  3. 代码

    a. java code

    b. python code

  4. 复杂度分析

    用S和P分别代表字符串和匹配模式的长度

    4.1 回溯法

    ​ 时间复杂度: O((S + P) * 2^(S + P/2))

    ​ 空间复杂度: O((S + P) * 2^(S + P/2))

    4.2 动态规划

    ​ 时间复杂度: O(T * P)

    ​ 空间复杂度: O(T * P)
