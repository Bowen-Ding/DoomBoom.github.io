岛屿数量(Number of Islands)

leetcode 第200题

  1. 题目描述

  1. 解题思路

    深度优先遍历: 利用递归来依次完成单个方向的查找,然后回溯继续查找

    广度优先遍历: 利用队列,将陆地不断加入队列,扩展周围连着的陆地加入队列,同时移除用于扩展的陆地

    并查集: 需要理解并查集类的特性,初始数量为单个元素的总和,把需要联合的元素(对应本题为所有水域联合,相连的陆地联合)依次相连最终指向同一个根节点,每联合一个元素,元素数量和减1,最终没有需要联合的元素时,就为岛屿数量加上1个水域

  2. 代码

    a. java code

    b. python code

  3. 复杂度分析

    时间复杂度: 均为O(m*n)

罗马数字转整数(Roman to Integer)

leetcode 第13题

  1. 题目描述

  2. 解题思路

    利用map/dictionary储存罗马数字和阿拉伯数字映射关系,遍历所给罗马数字,不断加和,对于特殊字符的处理有2种方式:

    ​ ①. 将特殊字符也加入map/dictionary,并优先判断

    ​ ②. 若当前字符的映射值小于后一个字符的映射值,则总和减去当前字符的映射值

    因为需要考虑特殊字符的情况,为了表面index溢出 遍历字符时只能到倒数第二位,然后单独处理最后一位字符;这里我将所给字符最后加了一个自定义字符,并规定自定义字符对应的阿拉伯数字为0,这样可以直接遍历到最后一位,也不用考虑所给字符只有1位的情况

  3. 代码

    a. java code

    b. python code

  4. 复杂度分析

    时间复杂度: O(n)

盛最多水的容器(Container With Most Water)

leetcode 第11题

  1. 题目描述

给定 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

示例:

输入: [1,8,6,2,5,4,8,3,7]
输出: 49

    a. 暴力法: 两层循环求取所有结果,找到最大值,代码略

正则表达式匹配(Regular Expression Matching)

leetcode 第2题

  1. 题目描述

  2. 解题思路

    a. 解法 回溯法: 运用递归,逐个匹配s的字符

    b. 解法 动态规划: 用一个二维数组保存 s[i:] 与 p[j:] 的匹配结果,返回s[0:] 与 p[0:] 的;这题动态规划有两种思路,一种从第一个字符往最后一个字符匹配,另一种从最后一个字符开始倒推,用java和python各实现一种

  3. 代码

    a. java code

    b. python code

  4. 复杂度分析

    用S和P分别代表字符串和匹配模式的长度

    4.1 回溯法

    ​ 时间复杂度: O((S + P) * 2^(S + P/2))

    ​ 空间复杂度: O((S + P) * 2^(S + P/2))

    4.2 动态规划

    ​ 时间复杂度: O(T * P)

    ​ 空间复杂度: O(T * P)
