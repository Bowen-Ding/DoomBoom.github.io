二叉树的后序遍历(Binary Tree Postorder Traversal)

leetcode 第145题

  1. 题目描述

  2. 解题思路

    递归法: 注意向结果list添加节点的顺序即可

    迭代法: 从根节点开始依次迭代，弹出栈顶元素输出到输出列表中，然后依次压入它的所有孩子节点，按照从上到下、从左至右的顺序依次压入栈中。因为深度优先搜索后序遍历的顺序是从下到上、从左至右，所以需要将输出列表逆序输出。

  3. 代码

    a. java code

    b. python code

  4. 复杂度分析

    迭代法:

    ​ 时间复杂度: O(n)

    ​ 空间复杂度: O(n)

    递归法:

    ​ 时间复杂度: O(n)

    ​ 空间复杂度: 树高,最坏为链表O(n),平均O(log n)

最长回文字子串(Longest Palindromic Substring)

leetcode 第2题

  1. 题目描述

  2. 解题思路

    2.1 中心扩展法

    ​ 遍历字符串,假设当前字符是回文字符串的中心,然后向外扩散,分为2种情况:1. 中心为单字符;2. 中心为2个相等字符。

    2.2 动态规划

    ​ 假设当前字符串为回文字符串,则其必然满足左边界等于右边界,且去掉2个边界依然为回文字符串,以此建立状态转移方程

  3. 代码

    a. java code

    b. python code

  4. 复杂度分析

    4.1 中心扩展法

    ​ 时间复杂度: O(n)

    ​ 空间复杂度: O(1)

    4.2 动态规划

    ​ 时间复杂度: O(n²)

    ​ 空间复杂度: O(n²)

leetcode 第2题

  1. 题目描述

  2. 解题思路

    通过一个变量carry保存每次运算的进位值,同时作为下次运算的初始值.

    a.因为需要不断的迭代计算,为了不破坏原始数据和结果集,需要对两个ListNode以及结果ListNode创建额外的ListNode对象引用;

    b.将两个ListNode对象的当前值以及carry值相加得到当前计算的总和,总和进位的值赋给carry,个位数为结果的当前值;

    c.将用于计算的引用对象的next值赋值给自己,用于结果的引用对象的next初始值设为carry赋给自己,重复b,直到两个计算对象的next值和carry均为空。

  3. 代码

    a. java code

    b. python code

  4. 复杂度分析

    时间复杂度: O(max(m,n))

  • 系统版本: centos7.0

  1. 关闭防火墙和SELINUX

    务必确保防火墙和selinux关闭,否则会导致许多问题

    核心是需要保持这几个地方一致:

    1. 宿主机适配器设置中VMnet8属性中IPv4属性设置:
      • 不自动获取IP地址
      • ip地址和网关在同一网段,这里我设置为192.168.13.101
      • 默认网关与首选DNS相同
    2. 虚拟机中网关与宿主机VMnet8默认网关相同,自定义IP处于同一网段,即都为1中示例的192.168.13.X,首选DNS与网关相同

    ​ c. 然后到github项目根目录新建一个名为CNAME的文件，文件内容填写上域名
