盛最多水的容器(Container With Most Water)

leetcode 第11题

  1. 题目描述

给定 n 个非负整数 a1，a2，…，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

示例:

输入: [1,8,6,2,5,4,8,3,7]
输出: 49

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/container-with-most-water
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。
​

  1. 解题思路

    a. 暴力法: 两层循环求取所有结果,找到最大值,代码略

    b. 双指针法: 从两端开始,每次只移动高度较小的一端,向高度较高的一端移动一格;因为较高的一端移动后必然面积变小,所以可以不必考虑.(剪枝思路)

    a. java code

    b. python code

    4.1 暴力法

    ​ 时间复杂度: O(n²)

    ​ 空间复杂度: O(1)

    4.2 双指针法

    ​ 时间复杂度: O(n)

    ​ 空间复杂度: O(1)

二叉树的后序遍历(Binary Tree Postorder Traversal)

leetcode 第145题

  1. 题目描述

  2. 解题思路

    递归法: 注意向结果list添加节点的顺序即可

    迭代法: 从根节点开始依次迭代，弹出栈顶元素输出到输出列表中，然后依次压入它的所有孩子节点，按照从上到下、从左至右的顺序依次压入栈中。因为深度优先搜索后序遍历的顺序是从下到上、从左至右，所以需要将输出列表逆序输出。

  3. 代码

    a. java code

    b. python code

  4. 复杂度分析

    迭代法:

    ​ 时间复杂度: O(n)

    ​ 空间复杂度: O(n)

    递归法:

    ​ 时间复杂度: O(n)

    ​ 空间复杂度: 树高,最坏为链表O(n),平均O(log n)
